# TheCuteCats
The Clean Architecture Android App


## *Features*
- Get a list of Cute Cats.
- Display of Categories on Drawer
- Load More Cats By Click on 'loadmore' Button



## *Libraries Used*
- LiveData
- ViewModel
- Hilt
- OkHttp3
- Retrofit2
- Coroutine
- Unit Test
- UI Test
